In-class Exercises

Felt, part 1

  1. Download the latest data from Inside Airbnb for NYC. You can view all available data on their site.
  2. Log in to your Felt account.
  3. Create a new map.
  4. Upload the modified CSV file (this will take a minute or two as the file is still likely large).
  5. Change the styles of the points:
    1. Style the points by value with the room_type column to make a categorized style.
    2. Style the points by value with the price column to make a graduated style.
  6. Get the public link for the map.
