Attribute Joins

  1. Download today's data.
  2. Open the ne_10m_admin_0_countries_lakes shapefile in QGIS.
  3. Open the CSV file in urban-population-world-bank in QGIS. Open it as a delimited text layer with no geometry. Ensure the 2012 and 2022 fields are imported as decimal numbers rather than text.
  4. Attribute join the urban population data to the countries. Use Country Code as the Join field, WB_A3 as the Target field.
