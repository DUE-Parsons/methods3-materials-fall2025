Assignment 6

When you join two or more datasets, you combine them feature-by-feature, and this often gives attributes to a feature that it would not have otherwise had.

Attribute joins add attributes to features based on a unique identifier. Often you will have a spatial dataset (such as a shapefile) and a CSV file with extra data that you would like to map. As long as both the spatial data and the CSV have a common unique identifier (sometimes this is a name, but it could be a unique number or an address) you can join them in QGIS and add the data from the CSV to the spatial data.

Use attribute joins and US Census data to map either renter occupancy or median household income (or some other census data if it's more relevant for your research) by census tract in the area within NYC you have been focusing on over the past few assignments.

Download census data using IPUMS NHGIS, join with the census tracts, select only the census tracts in NYC, and make a choropleth map with some of the joined data. Choose which data to get:

For any of the datasets you will want to do the following:

  1. Go to IPUMS NHGIS
  2. For Geographic Levels select CENSUS TRACT
  3. For Years select 2017-2021 under 5-year ranges

For renter vs owner-occupied under Topics select HOUSING > Tenure and Mortgage Status and select the first table (Tenure).

For household income under Topics select INCOME > Household and Family Income and select the table called Median Household Income in the Past 12 Months (in 2021 inflation-adjusted dollars).

For both datasets:

  1. Also select the Census Tract file under the GIS FILES tab.
  2. Click Continue in the Data Cart twice.
  3. Submit your request and wait for the data to be prepared.
  4. Download the Tables and GIS Files.
  5. Look at the codebook in the tables zip file to determine which columns you're interested in.
  6. Open the census tracts shapefile in QGIS.
  7. Open the CSV in QGIS. You do not need to open this file in Excel first.
  8. Join the CSV to the shapefile.
  9. Make a choropleth based on one or more columns you joined to the shapefile.
  10. Optionally clip or select by location to focus the map on NYC.
