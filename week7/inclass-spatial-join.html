In-class Exercises

Spatial Joins

  1. Open Council District Boundaries in QGIS.
  2. Open the CSV in the Inside Airbnb folder in QGIS.
  3. Search in the bottom left for Join attributes by location (summary).
  4. In this window select:
    1. Join features to: Council District Boundaries
    2. By comparing to: Inside Airbnb
    3. Fields to summarise: price
    4. Summaries to calculate: mean, median
  5. Make a choropleth with the result.

Optional additional spatial join

If you have time and want to apply geoprocessing with a spatial join:

  1. Open New York City Bike Routes.geojson in QGIS.
  2. Open Council District Boundaries in QGIS.
  3. Intersect the two files to get a bike routes file where the bike routes are clipped to the district boundaries. Use Vector > Geoprocessing > Intersection and select the two files, bike routes as input and council distrcts as overlay.
  4. Change your project's length units by going to Project > Properties > General and changing Units for distance measurement to miles.
  5. Use the field calculator to add a length field to the file.
  6. Search in the bottom left for Join attributes by location (summary).
  7. In this window select:
    1. Join features to: Council District Boundaries
    2. By comparing to: the intersected bike routes
    3. Fields to summarise: length (as calculated above)
    4. Summaries to calculate: sum
  8. Make a choropleth with the result.
